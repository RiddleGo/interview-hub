# interview-hub

面经合集 **导航站**：用浏览器访问下方 Pages 地址，可一键进入四个题库仓库与各自的单页 `study.html`（在线预览）。

## 在线访问

启用 GitHub Pages（见下）后，站点一般为：

**https://riddlego.github.io/interview-hub/**

（若使用自定义域名，以仓库 Settings → Pages 显示为准。）

## 首次启用 GitHub Pages

1. 打开本仓库 **Settings → Pages**。
2. **Build and deployment** → Source 选择 **GitHub Actions**。
3. 推送任意更新到 `main`，等待 **Actions** 里工作流变绿后刷新 Pages 地址。

## 本地预览

直接用浏览器打开根目录的 `index.html` 即可（相对链接在本地同样可用）。
