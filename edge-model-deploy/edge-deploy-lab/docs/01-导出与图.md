## 你要完成什么

把 `yolov5s.pt` 导出为 **静态 shape** 的 FP32 ONNX，并通过 checker 与 ORT 数值验收。

---

## 操作步骤

```bash
python scripts/01_export_onnx.py --config configs/project.yaml
python scripts/02_check_onnx.py --config configs/project.yaml
python scripts/03_preprocess.py --config configs/project.yaml
python scripts/04_ort_golden.py --config configs/project.yaml
```

Netron 打开 `deliverables/01_export/model_fp32.onnx`，确认输入 `images: [1,3,640,640]`。

---

## 必做练习（学中练）

### 练习 A：动态 shape（Q2）

```bash
python scripts/01_export_onnx.py --dynamic
python scripts/02_check_onnx.py --onnx deliverables/01_export/model_fp32_dynamic.onnx
```

Netron 里看到 `-1` 或动态维后，**改回静态**再进入阶段 2。

### 练习 B：Opset（Q3）

```bash
python scripts/01_export_onnx.py --opset 11
# 若 checker/atc 报错，记录原因，改回 project.yaml 中的 opset
```

## 面试口述要点

- **Q1**：`.pt` 绑 PyTorch；ONNX 是中间合同；`.om` 是已 lowering 的芯片二进制。ORT 是验收基准，不是交付物。
- **Q4**：ORT 通用 EP；NPU 白名单 + 编译期 fusion/tiling/layout，必须同输入双端对比。
