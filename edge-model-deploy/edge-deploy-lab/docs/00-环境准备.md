# 阶段 0：环境准备

**覆盖面试题**：Q19（NPU vs GPU）  
**深读**：[NPU与GPU部署差异_Pro.md](../../NPU与GPU部署差异_Pro.md)

---

## 1. Python 环境

```bash
cd edge-deploy-lab
pip install -r requirements.txt
python -c "import torch, onnx, onnxruntime; print(torch.__version__, onnx.__version__)"
```

## 2. 昇腾 CANN

```bash
# 路径以你机器为准
source /usr/local/Ascend/ascend-toolkit/set_env.sh

atc --help
amct_onnx --help   # 可选，量化用
npu-smi info       # 或 ascend-dmi
```

确认 `configs/project.yaml` 里 `soc_version` 与板子一致（如 `Ascend310P3`）。

## 3. 开发板 adb

```bash
adb devices
adb shell uname -a
```

## 4. 权重与示例图

```bash
mkdir -p deliverables/01_export
# 下载 yolov5s.pt 到 deliverables/01_export/
python scripts/00_make_sample_image.py
```

## 5. 验收

- [ ] `pip install -r requirements.txt` 无报错
- [ ] `atc` 在 PATH（无 CANN 可先跳过，用 `run_pc_smoke.py`）
- [ ] `yolov5s.pt` 已就位

## 6. 口述练习（Q19）

用一句话：**GPU 卖通用和大 batch 吞吐；NPU 卖低功耗下编译期优化后的确定性延迟，代价是算子白名单和迁移成本。**
