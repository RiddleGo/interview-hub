## 对比顺序

1. **FP16 om** vs ORT FP16，同 `input.bin`
2. FP16 过关后再比 **INT8 om**

```bash
INFER_BIN=/path/to/your/acl_main bash scripts/09_board_infer.sh fp16
python scripts/07_compare.py

INFER_BIN=/path/to/your/acl_main bash scripts/09_board_infer.sh int8
python scripts/07_compare.py
```

`infer` 调用约定（与 CANN sample 对齐）：

```
./infer <model.om> <input.bin> <output.bin> <shape>
# 例：./infer model.om input.bin output_npu.bin 1,3,640,640
```
